# Ajay | Full-Stack Developer (Portfolio)

A clean, responsive single-page portfolio built with **HTML + CSS + JavaScript**.

## Features

- Fixed navigation header with blur background
- Mobile hamburger menu
- Smooth scrolling between sections
- Projects shown as responsive cards
- Resume download button

## Project Structure

- `portfolio.html` — page markup
- `portfolio.css` — styling and responsiveness
- `portfolio.js` — mobile menu + scroll behavior
- `assets/` — images and resume PDF
- `Documentation.md` — detailed explanation of the components

## Setup / Run

This is a static site.

**Option A (quick):**

- Open `portfolio.html` directly in your browser.

**Option B (recommended):**

- Use any static server (e.g., VSCode Live Server) from the `frontend/portfolio` folder.

## Customization

- Edit text/content directly inside `portfolio.html`.
- Edit styles in `portfolio.css`.
- Add/remove projects by editing the `#projects` section cards.

## Contact Form Note

The contact form is currently UI-only (no backend submission implemented).
If you want to handle submissions, you can add a `submit` handler in `portfolio.js` and connect it to an API.
